# 橙子下载器 · 图标/Logo 设计规范

## 需要提供的图片（共 4 张）

| 文件名 | 尺寸 | 格式 | 形状 | 用于 |
|--------|------|------|------|------|
| `icon-512.png` | 512×512 px | PNG 不透明 | **圆角方形**（系统自动裁） | PWA 图标、安装到桌面 |
| `icon-192.png` | 192×192 px | PNG 不透明 | 同上，从 512 缩小即可 | PWA 通知图标 |
| `apple-touch-icon.png` | 180×180 px | PNG 不透明 | 同上 | iOS 添加到主屏幕 |
| `logo.png` | 256×256 px 以上 | PNG **圆形透明背景** | ⭕ 正圆形 | 头部 Logo（显示在 40×40 圆形容器中，超出部分被裁） |
| `icon.svg` | 矢量 | SVG | 圆角方形 | 浏览器标签、矢量图标 |
| `favicon-32.png` | 32×32 px | PNG | 同上 | 浏览器标签 |

## 关键说明

**头部 Logo（logo.png）**
- 显示在 40×40px（手机）/ 48×48px（桌面）的**正圆形**容器内
- 图片用 `object-cover` 居中填充，四角被裁
- **建议直接做成圆形透明 PNG**，这样不管容器怎么裁都不会有问题
- 最小尺寸：256×256，越大越清晰

**PWA 图标（icon-512 / icon-192）**
- 系统会自动裁成圆角方形或圆形（iOS 加圆角，Android 自适应）
- 重要内容放在中间 80% 区域，避免边缘被裁
- 不透明，需要品牌底色

**矢量图标（icon.svg）**
- 与 PWA 图标同一设计
- 512×512 viewBox

**所有图标必须是同一套设计**，仅尺寸和形状不同。
