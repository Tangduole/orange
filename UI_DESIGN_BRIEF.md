# 橙子下载器 UI 设计需求

## 产品概述
多平台视频下载工具，支持抖音/TikTok/X/YouTube等平台下载。

## 目标用户
- 内容创作者
- 电商从业者（视频带货）
- 自媒体从业者
- 音乐/配音爱好者

## 当前界面结构

### 1. 首页
- Logo + 标题
- 链接输入框
- 平台选择（自动识别）
- 下载选项（视频/音频/字幕）
- 下载进度显示
- 历史记录面板

### 2. 会员订阅页
- 当前等级显示
- Free vs Pro 对比
- 升级按钮
- FAQ

## 设计方向

### 配色
- 当前：深色主题 + 橙色点缀
- 品牌色：橙色系

### 风格
- 现代简洁
- 工具类应用，清晰易用

## 需要设计的页面

1. **首页** - 主下载界面
2. **会员订阅页** - 升级引导
3. **登录/注册弹窗** - 简洁表单

## 技术栈
- React + TypeScript
- Tailwind CSS
- 深色主题

## 参考
- Vercel 部署的 Web 版本
- Android APK 已可构建
